Abstract

Firstly, the basic calculator functions (Atom-Calculate) are implemented, such as addition, subtraction, multiplication and division with parentheses and mixed operations with arbitrary squares, and further functional operations on the resultant Ans, such as trigonometric functions, logarithmic functions, etc. Further, menu functions have been created and improved in the dialogs. In addition to the basic functions such as Help, contact us, etc., a new Molecule mode function has been added for more powerful arithmetic and mathematical functions, such as: solving linear equations, solving multiple equations in one element, matrix operations, complex number operations, date conversion, decomposition of prime factors, time conversion, simple integral calculations, multi-decimal conversion, etc. (Molecule- Calculate). Drawmode has been added, which can be used for drawing. In addition, Leisure mode has been added, linking several mini-games that are not fully encapsulated with the calculator system. Finally, in order to improve user-friendliness and security, the developed interface will be beautified, and user login passwords and other features will be added to improve security. The system inherits most of the features of the common calculator systems, especially in addition, subtraction, multiplication and division with brackets and mixed operations with arbitrary squares. The main innovation of this system is that it is not only limited to ordinary calculator programs, but also adds new functions in the mode Molecule-Calculate such as solving linear equations, solving multiple equations in one element, matrix operations, complex number operations, date conversion, decomposition of prime factors, time conversion, simple integral calculations, and multi-decimal conversions. Drawmode has been added, which can be used for drawing.
